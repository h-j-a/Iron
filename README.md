Iron
A minimal, offline-first workout tracker PWA. Log sets, build routines, and track your progress — all from your browser.
Try it live →
Features

🏋️ Workout Logging — Log sets with weight and reps, auto-filled from your last session
📋 Routines — Build and save custom routines with any exercises
⏱️ Rest Timer — Auto-starts after each set with audio, vibration, and notifications
🏆 PR Detection — Automatically flags personal records and estimates your 1RM
📈 Progress Charts — Visualise your weight progression per exercise over time
🍽️ Plate Calculator — Hints shown as you type so you know exactly what to load
⚡ Supersets — Group exercises together for superset-style training
💪 Bodyweight Support — Toggle any exercise to bodyweight mode
📦 Export / Import — Back up and restore your data as JSON

Tech Stack

HTML, CSS, JavaScript — single file app, no frameworks
localStorage — all data stored locally in your browser
Deployed on Netlify

Privacy
All your workout data is stored locally in your browser using localStorage. Nothing is sent to any server. Everything stays on your device.
